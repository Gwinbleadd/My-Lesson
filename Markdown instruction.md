## Списки

Что бы добавить не нумерованные списки, необходимые пункты нужно выделить звездочкой (*).
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Что бы добавить нумерованные списки, небходимые пункты выделить номером.
Например, вот так:

1. Элемент
2. Элемент
3. Элемент
